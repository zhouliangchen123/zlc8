实时时间

✨

玻璃态美学

采用玻璃态 (glassmorphism) 设计，搭配霓虹渐变与动态光影，视觉体验独特而优雅。

🌙

主题切换

内置深色 / 浅色双主题，一键切换，保护视力，适配不同场景与光线。

🚀 使用指南

1

千问 API 已预配置

已为你内置阿里云千问 API Key，开箱即用。如需更换，可在 js/config.js 中修改。

2

本地启动方式

在项目目录中使用任一本地服务器运行：
python -m http.server 8000 或 npx serve，然后在浏览器打开对应地址。

3

对话小技巧

• 在输入框按 Enter 发送，Shift+Enter 换行
• 点击快速建议按钮可一键填入示例问题
• 对话记录保存在本地浏览器，刷新不丢失

4

关于 CORS

浏览器同源策略可能阻止直接 file:// 打开调用 API。务必使用本地 HTTP 服务器运行本项目。
